
## Installation

First at all clone the repository with git:

```bash
git clone https://github.com/doneber/yt-summarizer
```

Install yt-summarizer with npm

```bash
  cd yt-summarizer
  npm install
```

### Development environment (backend)

```bash
  npm run dev:backend
```

### Production environment

```bash
  npm run build
  npm run start
```
